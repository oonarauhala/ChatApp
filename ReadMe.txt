Chat client retrieves messages from server and displays them with
recycler view. User must set a username before sending messages
and is able to change their username. Toast is displayed for the user
when username is changed, if username is not available and if username
is not set.
Folder ChatProject is for Server.
Folder ChatClient is for Chat Client.
